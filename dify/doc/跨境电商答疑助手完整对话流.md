## 2. 意图识别

![意图识别节点](../images/6.2/04.png)

- **代码实现**
  - **作用：** 判断用户是否属于打招呼用语、亦或者找人工、还是询问电商知识（需要 RAG 检索）

- **节点展开详情：**

![意图识别节点详情](../images/6.2/05.png)

- **作用：** 判断用户的 query 是否属于跨境电商知识范畴

  ```properties
  主要三种选择：打招呼(greeting)；找人工(human)；RAG检索(rag)
  ```

- **实现方式：** 利用《代码执行》节点完成

- **实现代码如下：**

```python
import random

def main(query:str)->dict:
    """
    基于固定短语匹配的跨境电商问答处理器
    """
    query = query.strip()

    # 打招呼相关的固定短语
    greeting_phrases = {
        "你好", "您好", "hi", "hello", "嗨", "哈喽", "哈罗",
        "早上好", "下午好", "晚上好", "你是谁", "你叫什么",
        "你好吗", "怎么样", "你在吗", "在不在",
        "你能干什么", "你会做什么", "你能做什么",
        "开始", "开始咨询", "测试", "试试",
    }

    greeting_response = "你好，很高兴为您服务！我是您的跨境电商学习小助手，专业为您答疑解惑。"

    # 人工服务相关短语
    human_service_phrases = {
        "人工服务", "人工客服", "转人工", "找人工",
        "真人服务", "真人客服", "联系客服", "找客服",
        "联系老师", "找老师", "投诉", "举报",
        "退款", "退货", "售后", "售后服务",
    }

    human_service_response = "同学，点击 https://www.123.com 可进入人工答疑"

    # 礼貌用语
    thank_phrases = {"谢谢", "感谢", "多谢", "thanks", "thank you", "谢谢你"}
    goodbye_phrases = {"再见", "拜拜", "bye", "goodbye", "88", "see you"}

    polite_responses = {
        "thank": [
            "不用客气，随时为您服务！",
            "很高兴能帮助到您！",
            "不客气，祝您跨境电商生意兴隆！"
        ],
        "goodbye": [
            "再见！期待下次为您服务！",
            "祝您生活愉快，有问题随时来找我！",
            "拜拜！有问题随时回来咨询！"
        ]
    }

    def normalize_text(text: str) -> str:
        import re
        cleaned = re.sub(r'[^\w一-鿿]', '', text.lower().strip())
        return cleaned

    def exact_match_check(query_text: str, phrase_set) -> bool:
        normalized_query = normalize_text(query_text)
        for phrase in phrase_set:
            if normalize_text(phrase) == normalized_query:
                return True
        return False

    query = query.strip()
    if not query:
        return {"type": "error", "response": "请输入您的问题。", "need_rag": False}

    if exact_match_check(query, thank_phrases):
        return {"type": "greeting", "response": random.choice(polite_responses["thank"]), "need_rag": False}
    if exact_match_check(query, goodbye_phrases):
        return {"type": "greeting", "response": random.choice(polite_responses["goodbye"]), "need_rag": False}
    if exact_match_check(query, greeting_phrases):
        return {"type": "greeting", "response": greeting_response, "need_rag": False}
    if exact_match_check(query, human_service_phrases):
        return {"type": "human_service", "response": human_service_response, "need_rag": False}

    return {"type": "rag_needed", "response": "", "need_rag": True}
```

## 6. 大模型 RAG 回答

![大模型RAG节点](../images/6.2/11.png)

- **作用：** 接受用户的 query 和上下文，经过大模型得到结果
- **实现方式：** 添加《LLM》节点
- **上下文：** 知识检索内容

**RAG 大模型系统 Prompt：**

```markdown
# 角色
你是一位专业且高效的跨境电商答疑小助手，精通跨境电商领域的各类知识。

## 重要参数
- 上下文内容：{{#context#}}

## 技能
### 技能 1: 基于上下文解答问题
1. 严格依据提供的上下文内容进行回答，不添加上下文中未出现的信息。
2. 回答需精准、简洁、有条理。

### 技能 2: 答案优化
1. 若检索出来的上下文答案比较简单，需进行优化，但不能改变核心内容。

### 技能 3: 上下文不足信息补齐
1. 若给出的上下文信息非常少或没有，需结合自身专业知识回答。

### 技能 4: 敏感词过滤
1. 遇到敏感词汇时，利用插件进行敏感词搜索并去除。

## 限制:
- 仅回答与跨境电商相关的问题。
- 优先以检索的知识库内容为答案。
```

**用户 Prompt：**

```properties
用户输入问题：{x}query
```

## 7. 大模型闲聊回答

![闲聊大模型节点](../images/6.2/13.png)

- **作用：** 接受用户的问题，直接基于大模型本身回答结果

- **实现方式：** 添加《大模型》节点

**闲聊大模型系统 Prompt：**

```markdown
# 角色
你是一位专业且亲切的跨境电商答疑小助手，拥有广泛的通用知识储备。

## 技能
### 技能 1: 精准解答跨境电商问题
1. 运用专业知识结合案例为用户解答。
2. 在回答结尾注明"以上答案仅供参考"。

### 技能 2: 耐心回应非跨境电商问题
1. 运用已有知识耐心回答。
2. 在回答结尾注明"以上答案仅供参考"。

### 技能 3: 合理引导不明确问题
1. 对含义不明确的问题，输出"我还在努力理解您的问题，请您详细描述后再来询问吧。"

## 限制:
- 交流内容围绕跨境电商学习及其他非电商知识范畴。
```

**用户 Prompt：**

```properties
用户问题：{x}query
```
